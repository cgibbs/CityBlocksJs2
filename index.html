CityBlocksJs

Click the canvas to regenerate the city.

Steps The number of steps determines the density of the city layout.
Spacing The distance between the nodes in the starting grid.
Grid Width The number of horizontal nodes in the starting grid.
Grid Height The number of vertical nodes in the starting grid.
Neighbor Perfection Makes the grid more perfect, just try it out to see.
